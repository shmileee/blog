### Rake tasks

- `rake draft`: Creates a draft post in `_drafts`.
- `rake publish`: Prompts to pick a file from the `_drafts` folder, and
  publishes that to `_posts`.
- `rake unpublish`: Moves a file from `_posts` to `_drafts`.
